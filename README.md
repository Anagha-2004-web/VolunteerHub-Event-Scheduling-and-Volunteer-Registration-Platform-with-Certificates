# VolunteerHub: Event Scheduling and Volunteer Registration Platform with Certificates

## 📌 Overview
VolunteerHub is a web-based platform designed to simplify event management and volunteer coordination. It allows organizers to post events, manage volunteer registrations, and generate certificates for participants. Volunteers can explore events, register for roles, and download their participation certificates.

---

## 🚀 Features

- 📅 Event Scheduling by Organizers  
- 🧑‍🤝‍🧑 Volunteer Registration for Events  
- 🔐 Basic User Authentication (Login System)  
- 📊 Organizer Dashboard to View Registrations  
- 📁 Export Volunteer Data as CSV  
- 🎓 Certificate Generation for Volunteers
- 🏆 Point-based reward system for volunteer participation  
- 📊 Leaderboard to track and rank volunteer performance  
- 📍 Attendance tracking mechanism for event participation
